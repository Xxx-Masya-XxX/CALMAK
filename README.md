# CALMAK

## Установка

чтобы установить зависимости необходимо установить uv 
https://docs.astral.sh/uv/

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

1-ый запуск приложения будет долгим так как будут устанавливаться зависимости
```bash
# Запуск приложения 
uv run python main.py
```

## Возможности

- Создание и управление канвасами
- Добавление объектов (фигуры, текст, изображения)
- Иерархия объектов (родитель-потомок)
- Настройка свойств объектов
- Экспорт в PNG
